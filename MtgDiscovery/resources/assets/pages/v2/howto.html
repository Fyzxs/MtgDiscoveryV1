How To Use

Navigate Collection

Get to a card

  • Hit TAB until card is selected
  • Click on a card

Next Card

  • On a card - TAB
  • On a card - 'Right Arrow Key'
  • Click on the next card

Previous Card

  • On a card - SHIFT+TAB
  • On a card - 'Left Arrow Key'
  • Click on the previous card

Next Row

  • On a card - 'Down Arrow Key'
  • Click on the next row

Previous Row

  • On a card - 'Up Arrow Key'
  • Click on the previous row card

Next Page This is for the Binder View only

  • Click the "NEXT" button
  • Hit the 'Space Bar'

Previous Page This is for the Binder View only

  • Click the "PREV" button
  • Hit the ',' (comma) key

Specific Page This is for the Binder View only

  • Click the appropriate page button

Update Collection

There's no visual indicators of what value of the update has been entered.

Add Card(s)

  1. Navigate to a card
  2. Type the number of cards to add
  3. Hit Enter
Remove Card(s)
  1. Navigate to a card
  2. Type the minus sign (-)
  3. Type the number of cards to remove
  4. Hit Enter
Clear Entered Value
  • Navigate away from the card
  • Hit 'Delete' key
  • Hit '.' on the number pad
Note: Delete is not Backspace

Controls

Sort

  • Sets
    • Alpha
      • Displays the sets alphabetized by the set name.
    • Release (DEFAULT)
      • Displays the sets by their release date. This is limited to month/year.
    • Size
      • Displays the sets by the number of cards in the set.
    • Collector Only
      • Set %
        • Displays the sets by the percentage of completion.
      • Cards
        • Displays the sets by number of cards of the set owned.
  • Cards
    • Alpha
      • Displays the cards alphabetized by the card name.
    • Card #
      • Displays the cards by their set number.
    • Price
      • Displays teh cards by their latest price. This information is updated periodically.

Order

  • Ascending
    • Sorts the items in ascending order of the sort selection. This is default for card views.
  • Descending
    • Sorts the items in descending order of the sort selection. This is default for set views.

Filter

The "Set Name" and "Card Name" text box can be quickly gotten to by hitting "SHIFT" twice in quick succession.
  • Sets
    • Collection Status
      • This shows for sets, and filters based on the collection status. Completed Sets, Partial Sets, Not Started Sets.
    • Set Name
      • Instead of using the browser search, the sets can be searched via partial patch of the name or set code.
  • Cards
    • Rarity
      • Allows filtering out cards of specific rarities.
    • Card Count
      • Allows filtering cards with a certain amount collected. Mostly useful to me for seeing cards I have none of.
    • Card Name
      • Filter cards down by partial match of the card name. This is useful when you have a few cards to enter and don't want to scroll.
  • Binder View
    • Pages
      • You can navigate around the pages of the virtual binder.

DANGER ZONE
The danger zone is useful controls that aren't easy to undo. While it's not heavily populated, it's important to keep away from the other controls.
This is the only control group that starts collapsed. Currently there's just one control here.
Add one of each visible
This will add 1 card of each visible card on the page. It's useful for known complete sets, or the binder view to add a full page.

Valuable controls... dangerous controls.

Pages

For any of the pages showing set or card boxes, if there is a collector id in the url, that users collection information will be included in the page.


Sets
This page shows my definition of "sets". I look at sets as separate things I can collect. If I decide I'm collecting "X", what is it I ACTUALLY have to collect. More importantly, what am I NOT feeling I have to collect? That's how I split these out. This is why a set like Zendikar Rising has 19 different sets associated with it. Any one of those I could collect in "isolation". That's not entirely true. I have to start with the core set. Everything else is then an option. This might change via this site as I get close to the smaller sets.

Set Cards

Collector Summary

9 card pages
The way I collect cards is in 9 card binder pages, so I show it.

Binder View
This displays the set as 9 cards at a time, corresponding to the 9 card sheets for binders.
The display like this has been super helpful for me while going through my collection to ensure I have all the cards and they are organized correctly. It also makes knowing what links to click to be redirected to a store for purchasing really easy.
A fun feature I added after some purchasing mistakes is non-owned cards will be transparent. When I binder cards, I put a non-set land in backwards. This makes a quick visual indicator the card is not there. The transparency showing the back of the card is because that's what I do. It helps me line up my actual binder page to the page on the screen.
